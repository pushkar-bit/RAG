this is multi-tenant Retrieval-Augmented Generation (RAG) platform that enables organizations to upload, index, and search their internal documents through an intelligent conversational interface. The system processes documents, generates vector embeddings, retrieves relevant knowledge, and provides accurate, context-aware answers grounded in organizational data rather than relying solely on general AI knowledge
I created it to help students, corporates and  organizations with a centralized AI-powered knowledge assistant that can efficiently retrieve, understand, and answer questions from their internal documents. By combining document processing, vector search, and large language models, the platform reduces the time spent searching for information, improves knowledge accessibility, and delivers accurate, context-aware responses grounded in trusted organizational data.
I used Next.js,TypeScript, Tailwind CSS, Zustand, TanStack Query for frontend, Node.js, Express.js, TypeScript, BullMQ for backend and MongoDB for database along with groq api key for the AI.
